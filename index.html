ScoreTRACKER

A tool to help you manage your scores for any game

Welcome to scoreTRACKER

A tool to track the scores of your games. Mobile friendly, to have it always with you.

Darts mode

The most common way to play darts is to set an initial score of 301 or 501, and then remove points at each round until one player reaches zero.

For this reason, when enabled, the darts mode will subtract points instead of adding them, and the leaderboard will show the final score from the lower, to the higher.

How to play

Click on new game, and choose if you want to enable or disable Darts mode.

You will need to select an initial score, and you have the option to set a limit.
If the limit is set, you will be prompted to finish the game when one player reaches or exceeds the limit.

If you enable darts mode the limit is automatically set to 0.

Click on start to begin.

Add Players

Click on the Add players icon to start add each player.

You can select the name and the initial score will be automatically set based on the one set for the game.

You can edit each player's initial score when you add a new player, so that you can add players in the middle of the game, and choose for how many points they can start.

Finish game

If you click on the finish button, or if one of the players reaches the limit, you will be prompted to finish the game.

Click on finish to see the leaderboard and the winner of this round.

Reset score

If you want to start a new round you can click on Reset score, this will keep the players you have created, and will set the score for all players to the starting score chosen for the game.

Delete players

You can click on the delete button next to each player to remove them from the game.
